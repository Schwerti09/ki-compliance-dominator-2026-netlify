# KI Compliance Dominator 2026 (Netlify-ready)

This is a **deploy-ready** Next.js 15 (App Router) project scaffold.

## Local dev

```bash
npm install
npm run dev
```

## Netlify

- Build command: `npm run build`
- Publish directory: `.next`
- Netlify plugin: `@netlify/plugin-nextjs`

Environment variables: copy `.env.example` values into Netlify Site settings.

## Notes

This repo contains **stubs** for marketing pages, dashboard, and a health endpoint.
